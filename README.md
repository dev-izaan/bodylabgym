# Body Lab Gym — Premium Performance Fitness

Abu Dhabi's premier destination for high-performance training. Built with **Next.js 14**, **Tailwind CSS**, and **GSAP**, this platform embodies the elite boutique gym experience.

## 🚀 Experience
**Live Site:** [https://bodylabsgym.web.app](https://bodylabsgym.web.app)

## 💎 Features
- **Elite Aesthetic**: A bespoke glassmorphism design system tailored for UAE high-performance studios.
- **Dynamic Interactions**: Magnetic navigation components and smooth scroll-reveal animations powered by GSAP.
- **Unified Theme Control**: Seamlessly switch between high-contrast light and dark modes with integrated state persistence.
- **Full Responsiveness**: Optimized for high-resolution deskops and high-performance mobile devices.
- **Optimized Performance**: Hardware-accelerated transitions and optimized assets for smooth exploration.

## 🛠️ Technical Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS & Vanilla CSS (Custom PillNav)
- **Animations:** GSAP (GreenSock)
- **Icons:** Lucide React
- **Hosting:** Firebase Hosting (Next.js SSR support)

## 📦 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/dev-izaan/bodylabgym.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
